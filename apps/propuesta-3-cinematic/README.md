# Propuesta 3 — Cinematográfico Inmersivo (Dark)

Estética **oscura, dramática y premium**, pensada para causar impacto.

- **Tipografía:** Space Grotesk (display) + Inter (texto).
- **Paleta:** fondo casi negro, texto claro y un gradiente de acento (violeta → cian). Contraste WCAG AA.
- **Layout:** hero a pantalla completa con gradientes animados, tarjetas con *glassmorphism*.
- **Animación:** barra de progreso de scroll, *blobs* animados, entradas con desenfoque y parallax (Framer Motion).

## Desarrollo

```bash
pnpm install
pnpm --filter propuesta-3-cinematic dev   # http://localhost:3003
```

Sin Supabase, usa datos de ejemplo. Ver README raíz para Supabase y despliegue en Vercel
(Root Directory = `apps/propuesta-3-cinematic`).
