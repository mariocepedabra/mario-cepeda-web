# Propuesta 4 — Cálido / Orgánico Humano

Estética **cercana, cálida y humana**, con personalidad y formas suaves.

- **Tipografía:** Baloo 2 (display redondeada) + Nunito (texto). Amable y legible.
- **Paleta:** crema cálida, terracota, mostaza y verde salvia. Sensación acogedora.
- **Layout:** formas orgánicas (blobs), tarjetas muy redondeadas, sombras suaves, foto con recorte tipo blob.
- **Animación:** blobs flotantes, entradas con rebote suave (Framer Motion), microinteracciones amables.

## Desarrollo

```bash
pnpm install
pnpm --filter propuesta-4-organic dev   # http://localhost:3004
```

Sin Supabase, usa datos de ejemplo. Ver README raíz para Supabase y despliegue en Vercel
(Root Directory = `apps/propuesta-4-organic`).
