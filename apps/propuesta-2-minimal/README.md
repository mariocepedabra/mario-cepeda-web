# Propuesta 2 — Minimalista Suizo

Estética de **diseño suizo / tipográfico**: orden, claridad y foco absoluto en la tipografía.

- **Tipografía:** Inter (grande, tracking ajustado) + JetBrains Mono para etiquetas y numeración.
- **Paleta:** blanco, negro y un único acento (azul cobalto). Monocromo deliberado.
- **Layout:** grid estricta de 12 columnas, mucho espacio en blanco, filetes finos, secciones numeradas.
- **Animación:** microinteracciones sutiles (fades cortos, hover con desplazamiento, grayscale → color).

## Desarrollo

```bash
pnpm install
pnpm --filter propuesta-2-minimal dev   # http://localhost:3002
```

Sin Supabase, usa datos de ejemplo. Ver README raíz para Supabase y despliegue en Vercel
(Root Directory = `apps/propuesta-2-minimal`).
